# JobFlow 前端

基于 Next.js 15 + React 19 的对话式求职助手界面。

## 技术栈

- **Next.js 15** (App Router)
- **React 19**
- **TypeScript** 严格模式
- **Tailwind CSS v4**
- **Zustand** 状态管理
- **SSE** 流式通信

## 功能

- 💬 对话式求职助手（SSE 流式输出）
- 📄 简历管理（上传/解析/预览）
- 📊 投递进度看板（Kanban 风格）
- 🛠️ 技能库浏览

## 启动

```bash
npm install
npm run dev
# 访问 http://localhost:3000
```

确保后端运行在 `http://localhost:8001`（API 请求会自动代理）。
