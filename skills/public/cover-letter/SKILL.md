# Cover Letter 撰写技能

## 工作目标

为求职者撰写能打动招聘官的个性化求职信，体现候选人与职位的独特契合。

## 工作流程

### 步骤 1：分析候选人

解析简历，提取：
- 最突出的 3 个技能/经验
- 与目标职位最相关的项目

### 步骤 2：分析目标岗位

解析 JD，识别：
- 公司的核心痛点
- 该岗位最看重的能力
- 公司文化关键词

### 步骤 3：研究公司背景

（如有搜索工具）搜索目标公司的近期动态、产品、文化，寻找个性化素材。

### 步骤 4：选择角度

确定 Cover Letter 的核心主题：
- 技术专家角度（适合技术岗）
- 业务驱动角度（适合产品/运营岗）
- 成长潜力角度（适合应届/转行）

### 步骤 5：撰写三段式 Cover Letter

**第一段（引言）**：建立连接，说明为什么是"你 + 这个公司/职位"
**第二段（能力展示）**：2-3 个具体例子，聚焦价值输出
**第三段（结尾）**：表达热情，发起行动号召

### 步骤 6：双语输出

- `cover_letter_zh.md`：中文版
- `cover_letter_en.md`：英文版（自然翻译，非机械直译）
- `cover_letter_notes.md`：写作思路说明

## 注意事项

- 避免套话："我对贵公司非常感兴趣"→ 用具体的公司细节替代
- 长度控制：300-400 字（中文），250-350 词（英文）
- 每封信都应该是独一无二的
